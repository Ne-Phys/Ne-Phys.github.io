Luogu P1562 还是N皇后

  2020-03-06   题解   位运算 搜索  评论 字数统计: 247字 阅读时长: 1分

链接

摘要

   $n$皇后问题

   $0<n<=14$

解

   据数据范围 直接搜索会$tle$    使用位运算可以达到优化的目的    mxstt 即为列放置最终状态

   构造函数 dfs(cstt,l,r,ln)    递归边界 cstt==mxstt

  • k=~(cstt|l|r|ln)&mxstt 以获取改行状态 lowbit(k) 以列举可放置列
  • cstt 表示列放置状态 $0$可放置 $1$不可放置 cstt|lowbit(k) 以转移
  • l r 分别表示左斜 右斜的状态 $01$义同上 (l|lowbit(k))>>1 (r|lowbit(k))<<1 以转移
  • ln 表示当前行数 ln+1以转移

   即 dfs(cstt|lowbit(k),(l|lowbit(k))>>1,(r|lowbit(k))<<1,ln+1)

debug

  1. 设置一个$n$位二进制数的第$k$位为$1$  x|=k x|=1<<n-k
  2. l r 的转移 (l|cstt)>>1 (l|lowbit(k))>>1
  3. 尤其注意 maxstate 限制二进制位数

代码

#include<cstdio>
#include<cstdlib>
#define ifor(i,a,b) for(int i=(a);i<=(b);i++)

int n,ans,mxstt;
int stt[20];
char s[15];

inline int lowbit(int x)
{
    return x&-x;
}

inline void dfs(int cstt,int l,int r,int ln)
{
    if(cstt==mxstt) {ans++; return;}
    int k=~(cstt|l|r|stt[ln])&mxstt;
    while(lowbit(k))
    {
        dfs(cstt|lowbit(k),(l|lowbit(k))>>1,(r|lowbit(k))<<1,ln+1);
        k^=lowbit(k);
    }
}

int main()
{
    scanf("%d",&n);
    mxstt=(1<<n)-1;
    ifor(i,1,n)
    {
        scanf("%s",s+1);
        ifor(j,1,n)
            if(s[j]=='.')
                stt[i]|=(1<<n-j);
    }
    dfs(0,0,0,1);
    printf("%d",ans);
    system("pause");
    return 0;
}
